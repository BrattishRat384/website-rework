I3wm rice / config | 20-Apr-2025

I've been reconfiguring my desktop with a fresh install of Arch Linux.

I3wm was my choice of window manager as it was easy to config and simple to use

The config files are avalible on my github.

I picked rofi, feh, picom, kitty, polybar, dolphin, i3lock, pactl and playerctl as these all work well across devices and are also easy to config.

Making the configs easy to install and setup as most are modified defaults.

Choosing a light distro like ArchBTW and pairing it with I3 makes for a fast and responsive system.

Using arch and I3 means that my system sits at low cpu and ram usage - at idle sub 2% cpu and about 1gb of ram usage.
